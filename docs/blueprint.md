## Core Features:

- Hero Section: A visually striking introduction with name, profession, impactful tagline, and a clear call-to-action button. Use of a large image which can be switched using generative AI to assess user preferences
- About Me: A concise professional biography, optional professional photo, and elegantly presented key skills/tools with icons.
- Portfolio Grid: A clean grid showcasing recent projects with high-quality thumbnail images.
- Project Detail Page: Dedicated pages for each project with a gallery, detailed description of the challenge, solution, and results.
- Contact Form: A simple contact form with fields for name, email, and message.
- Social Links: Clear links to professional social media profiles (LinkedIn, Behance, etc.).
- Email Display: Clearly displayed email address for direct contact.

## Style Guidelines:

- Primary color: Brunswick green (#2D4739) for vibrancy and modern feel.
- Background color: Light gray (#F0F0F0) for a clean, neutral base.
- Accent color: Soft cyan (#7FFFD4) to complement the primary blue while offering good contrast for interactive elements.
- Body font: 'PT Sans', a humanist sans-serif for clean readability in body text.
- Headline font: 'Space Grotesk', a techy sans-serif, complementing the 'PT Sans', for a bold modern look.
- Clean grid-based layout with ample white space to direct user attention.
- Subtle hover effects on portfolio images and buttons for added interactivity.
